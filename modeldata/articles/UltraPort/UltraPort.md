# UltraPort

UltraPort 最早随 **ThinkPad T20** 推出，是一种专有 USB 1.1 端口，安装在带有该接口的机型的显示屏卡扣之间。它是一种非常短命的尝试，在 **T30** 推出后便消失了——因为 UltraPort 的配件远不如 UltraBay 的配件受欢迎或实用。

## 概述

由于是专有端口，UltraPort 的通用性不如普通 USB 端口。不过幸运的是，UltraPort 本质上只是一个带有自定义连接器的 USB 端口，因此可以通过将 USB 连接器焊接到其上来改造，使之能接受标准 USB 设备。但操作时需格外小心，因为 UltraPort 连接器直接集成在视频排线中。

### 配件

- UltraPort 蓝牙适配器
- UltraPort 摄像头 & UltraPort 摄像头 II
- UltraPort Compact Flash 适配器
- UltraPort 数字阵列麦克风
- UltraPort 红外适配器

### Linux 支持

UltraPort 会被内核的 USB 子系统自动检测并使用。关于其 Linux 支持状态，请参阅各配件的独立页面。

## 采用此技术的机型

- A20p, A21p, A22p, A30p, A31p
- T20, T21, T22, T23
