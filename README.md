# ThinkPad Database

本项目是一个简单的数据收集项目，目前收集了部分ThinkPad的关键配置和参数信息，这样用户可以不用每次查资料都去联想的PSREF翻阅故纸堆。

本项目也提供了一些PSREF中没有的机器的信息，比如周年纪念机和PRC Only的机型。

关于这块如果你有勘误或者线报，欢迎提交Issue或者Commit。

同时我们也提供了一个板块来供大家撰写自己想告诉别人的有关ThinkPad的小知识，欢迎大家投稿。

最近加入了一堆旧机器，因为人力有限，资料混乱和校正时脑袋不太清醒，可能有大量错漏（包括但不限于漏机器，参数写错，忘写组件的json等），烦请各位踊跃订正，在此鞠躬。
